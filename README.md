# TextHaze
It is a tool in C programming language which provides security and obfuscation of text files.

## Tech Stack
- C



## Features
- Create an encrypted file from software itself.
- To encrpt an existing file,the user can pick any of the three methods (monoalphabetic Cipher, Caesar Cipher, Rail Fence Cipher).
- For file decryption, the software will recognise the encryption method based on hash value and decypt accordingly.
